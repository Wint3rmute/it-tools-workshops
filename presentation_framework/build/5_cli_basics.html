Strumienie, przekierowania, pipe'y

Output redirection


  • komenda > plik - zapisz do pliku wynik komendy

  • komenda >> plik - dopisz do pliku wynik komendy

Pipe '|'


  • Wysyła output jednej komendy do drugiej

  • Może być łączone w łańcuchy

Przekierowanie inputu do programu


program <<< input


Dodatkowo:

program < plik_z_inputem
